# Hellfire Ledger Resolver — Frontend

Welcome to the **Hawkins National Laboratory Secure Interface**. This is a high-performance, retro-themed React application designed for ledger resolution, data transparency, and administrative token management. Inspired by the aesthetic of **Stranger Things**, it combines 1980s computer terminal vibes with modern engineering.

---

## 🗝️ Core Features

### 📡 Immersive Security Portal
- **Dual-Role Authentication**: Distinct login profiles for **Standard Agents** and **Administrators**.
- **System Control**: Integrated "RESTART UNIT" functionality for gated system reboots.
- **Personalized Welcome**: Custom greet sequence and dimensional clearance verification.

### 🍱 Role-Based Dashboards
- **Agent Desktop**: Full suite of analytical tools including Ledger Uplink, Debt Analytics, and Settlement Queue.
- **Admin Command**: Specialized dual-icon interface focused on protocol management (Token Minting & Token Sending).

### 📂 Advanced Data Processing
- **Uplink Intelligence**: 0-100% real-time progress bar for CSV uploads with "Abort Sequence" capability.
- **Ledger Intelligence**: Transparent "Real Data" table view for AI-cleaned transactions before optimization.
- **Settlement Queue**: Advanced filtering for optimized transactions (Mine, All, Settled, Remaining).

### 📊 Intelligence Modules
- **Debt Analytics**: Live metrics dashboard showing network savings, volume distribution, and top-tier debtors/creditors.
- **Mission Control Terminal**: Controlled background terminal that broadcasts real-time system logs and process diagnostics.
- **Admin Control**: Secure modules for Minting and Sending HFG (Hellfire Gold) tokens.

## 🚀 Getting Started

### 1. Prerequisites
Ensure you have **Node.js** and **npm** installed.

### 2. Installation
```bash
# Navigate to the frontend directory
cd frontend

# Install dependencies
npm install
```

### 3. Execution
```bash
# Start the development server
npm run dev
```

### 4. Integration
Ensure the **Hellfire AIML Backend** (Flask) is running on `http://127.0.0.1:5050` to enable full data resolution and token functions.
